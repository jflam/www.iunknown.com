Exchange Server Full-text indexing rocks

I always dreaded searching my email using Outlook. The search feature is so incredibly slow. While migrating my Exchange server to a beta of Exchange Server 2003 (I really wanted the new web outlook feature) I discovered the full-text indexing option. After enabling it, my searches through my 700MB mailbox are much faster; not instantaneous as I might have expected, but more than good enough for my purposes.
