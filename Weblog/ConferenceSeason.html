Conference Season
2003-01-21

So far, it's shaping up to be an interesting conference season.

I'm speaking at this year's WinDev conference June 16-20 in Boston.

I'm the C++ / C# track chair for the DevConnections conference May 6-9 in New Orleans. I'm really proud of the speaker lineup that I have put together for my tracks this year.

In the C++ track, I have Scott Meyers, Angelika Langer and Richard Grimes. Here's a list of the talks that they will be presenting:

Scott Meyers:

  1. Effective STL Programming pre-conference tutorial
  2. Approaches to the Static Initialization Problem
  3. rvalues in C++
  4. False Friends in C++

Angelika Langer:

  1. The Pragmatic Template Programmer
  2. Programming with C++ Exceptions
  3. Type Conversions in C++

Richard Grimes:

  1. Managed Extensions for C++
  2. Interop and Managed C++
  3. ATL7 Architecture

In the C# side track, I have Jason Clark, Aaron Skonnard, Bob Beauchemin, and Ingo Rammer. Here's a list of the talks that they will be presenting:

Jason Clark:

  1. A Day of Managed Programming with C#: pre-conference tutorial together with yours truly
  2. Introduction to .NET Code Access Security
  3. .NET Multithreaded Programming
  4. Dissecting the .NET Type System

Aaron Skonnard:

  1. The XML "Top Ten"
  2. Understanding .NET Web Services
  3. Mapping XML to Objects: Type System Mismatches

Bob Beauchemin:

  1. Optimizing SQL Server access in .NET
  2. Optimistic Concurrency choices using SqlClient
  3. ADO.NET and ASP.NET Applications

Ingo Rammer:

  1. Design and Development of Distributed .NET Applications Parts 1 and 2
  2. Why you should care about Enterprise Services

John Lam:

  1. Pragmatic C# development
  2. The CLR "Top Ten" (title shamelessly stolen from Aaron's talk :)

I'll be giving the following talks at WinDev:

  1. From AOP to Generative Programming: Creating Manageable Abstractions
  2. Using NAnt and NUnit to Manage the Development Process
  3. Web Forms Internals
  4. ASP.NET Debugging Technqiues
