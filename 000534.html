RSS / Atom feed parser in Python

Some random surfing led me to the Universal Feed Parser that's written in Python. When you discover a new piece of code, you immediately ask: what can I do with this?

Go check out the home page for the Universal Feed Parser. Marvel at the amazing amount of information that is captured via an interactive session with the feedparser and the associated comments. Contrast this with the corporate Amazon Web Services start page.

Make it as easy as possible to adopt your API and you'll greatly increase your chances of it happening. Don't make me dig.
