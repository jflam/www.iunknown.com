NUnit V2.2 bug with nullable types

I'm spending some time this evening with nullable types, and I discovered that NUnit V2.2 doesn't handle nullable types correctly. Here's a test case that fails:


[Test, ExpectedException(typeof(AssertionException))] 
public void NUnitAssertEqualBugWithNullableTypes() {
  int? x = null;
  Assert.AreEqual(null, x);
}
