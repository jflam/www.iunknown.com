Wintellect folks blogging

I'd like to announce that two other Wintellect folks are now blogging. Prolific writer extraordinaire, Dino Esposito has been running a blog for a couple of months now. I've always been in awe of how much writing Dino does. I suspect that there are actually several clones of Dino who do all that work. Also, Peter DeBetta has now started a blog. Peter's focus is on SQL Server, and he's got an article in the latest MSDN Magazine on creating managed UDT's in SQL Server Yukon.
