Fun with CodeDom

I decided to write a simplified version of Jeff Keys' most excellent Snippet Compiler to facilitate some experiments that I've been working on today. I absolutely love how easy it is to compile in-memory assemblies using System.CodeDom and friends:


private CompilerResults Compile()
{
  CodeDomProvider provider = new CSharpCodeProvider();
  ICodeCompiler compiler = provider.CreateCompiler();
  CompilerParameters parameters = new CompilerParameters();
  parameters.WarningLevel = 4;
  parameters.GenerateInMemory = true;
  parameters.GenerateExecutable = false;
  parameters.ReferencedAssemblies.Add("System.Web.Services.dll");
  parameters.ReferencedAssemblies.Add("System.Xml.dll");
  parameters.ReferencedAssemblies.Add("System.dll");
  return compiler.CompileAssemblyFromSource(parameters, textBoxSource.Text);
}
