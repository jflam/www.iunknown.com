A switch that I would love to see

is Mac OS X running on standard PC hardware. Dan Gillmor wrote this op-ed piece on the weekend.


I think that Apple needs to increase its user base at any cost or risk becoming irrelevant. The price / performance gap between Apple and PC hardware is only getting bigger these days.


I'm fortunate enough that I have the money to pay nearly $5K CDN to "switch" to OS X. Personally I would rather have spent $200 CDN on Mac OS X to run on one of my existing Windows boxes. I know that lots of other folks would love to be able to do the same thing too.


I suspect, however, that the biggest obstacle to running OS X on arbitrary PC hardware is driver support. That isn't going to be an easy problem for Apple to overcome, considering the bewildering array of hardware available in the PC space.
