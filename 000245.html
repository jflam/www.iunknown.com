Demos from my CLR Top 10 talk at VSConnections

Here's the demos and sample code from my CLR Top 10 talk at VS Connections today. Highlights of the samples include:



  • Using Reflection.Emit to create a dynamic in-memory assembly
  • Creating a multi-module assembly using VB and C# .netmodules
  • A HelloWorld app that uses DebugBreak to trigger a breakpoint in WinDbg so that you can see the generated release code
  • JitCost revisited

I had a great time giving the talk, and I hope everyone had fun there too!
