Infrastructure Stories Part 1

I've been a big fan of SpamBayes for some time now. Until recently, I've had a desktop computer running Outlook constantly so that SpamBayes can filter the incoming email off of my Exchange Server.

In the process of building some new infrastructure (more on this later) I've decided to consolidate servers. So I now have Outlook installed on my Exchange Server, and I use a disconnected RDP session to run Outlook and the Outlook SpamBayes plug-in. Works like a charm, and I have one less computer consuming power in my house.
