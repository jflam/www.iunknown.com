Virtual PC

To celebrate the new year, I decided that I needed to get Windows XP up and running on my TiBook. I went out and bought a copy of Virtual PC 6 , and had it up and running in about 5 minutes. A few hours later, I had Windows XP SP 1 + all of the current patches installed.


I even got ClearType enabled on a Windows XP session on my TiBook! It really is too bad that OS X doesn't have native support for ClearType. Hopefully Apple can license ClearType from MS (they have to since ClearType is heavily patented by Microsoft). My display looks a bit better after I followed Steve Makofsky's suggestion to make sure that font smoothing is turned on for LCD panels.


I'm totally amazed by the technology behind Virtual PC. Even with the recent performance bump provided by Version 6, the performance is still fairly slow, so I doubt that I'll be getting rid of my T23 anytime soon :) I haven't done any benchmarks yet, but I will once I get VS.NET up and running on my TiBook.
