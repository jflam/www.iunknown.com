It's here!

My Inspiron 8600 showed up today. After tearing open the package, I discovered to my horror that there isn't a trackpoint device (aka eraser head). I guess I had assumed that Dells would have a dual pointing device, but I was wrong. Fortunately, I've adapted rather well to the touch pad (much to my surprise) and I'm not hating it quite as much as I thought I would. The only thing that I don't like is having to shift my hand to reach the touchpad. With the eraser, your index finger is positioned exactly where it needs to be without requiring reposition your hand.

I immediately repaved using Windows XP Pro, and got my baseline image (Office + Virtual PC) installed. I'm copying the April MSDN Library onto the host, and will re-install MSDN in my guest OS's.

The keyboard takes a bit of getting used to - it's a different layout from the Thinkpad. I miss the groupings of the function keys, and the Windows key is positioned exactly where the Alt key is on the Thinkpad. Overall, however, the action is pretty good - about what I had expected.

The display is incredible. I'm blown away by the amount of real estate that I now have. I'm still running fonts at 96ppi, and scaling them appropriately for applications that I care about (command prompts + TextPad + VS.NET). I don't mind having the menus etc. take up less space with the 96ppi fonts - I treat them almost like icons - they're not things you generally read.

One thing that I really like is the improved scrolling speed for ClearType-rendered fonts. My ThinkPad and it's crappy 8MB S3 graphics card was horrible at scrolling HTML help rendered using ClearType.

I like the fact that the USB 2.0 port can power my external hard drive. On my T23, I had to connect to a USB card AND the USB 1.1 port for power.

I'll have more impressions once I'm done installing software.
