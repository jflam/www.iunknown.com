Hello SpamNet

Since I couldn't get Spam Assassin Pro running (and I needed to retire that other computer) I decided to try out CloudMark's SpamNet. Apparently this software works by collecting information about Spam in a giant distributed network - it relies on Outlook users everywhere to generate its "blacklist".


I had a tough time installing it though. Since I no longer run as a local administrator on my computer, I rely on switching to a shell with admin privileges to run setup programs. However, SpamNet's setup apparently uses your logon token to determine which account should be setup for SpamNet. Therefore my usual tricks didn't work. I had to manually add myself to the local admins group on my computer, install SpamNet, and then remove myself from the local admins group. This would be useful as a KB article on their web site.


Update: You also have to configure ISA server to open up port 2703 for outbound TCP traffic.
