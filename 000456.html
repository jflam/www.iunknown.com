Fritz is an old-school writer

I'm amazed that Fritz Onion still writes using pen and paper!

My current writing system is fairly minimalist - I use a Textile-style simplified markup language (not paying that angle bracket tax!) that is processed through a set of Perl scripts that generate HTML documents. HTML will replace PowerPoint eventually, and my Perl scripts will eventually generate either PDF (via XSL:FO) or XAML. I like my writing system because I no longer worry about formatting when creating content.

It's about eliminating mental friction, which is why I think Fritz writes with pen and paper. However, I still like being able to get stream of consciousness thoughts into a document by typing.
